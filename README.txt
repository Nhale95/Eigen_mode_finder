# Drawing Eigenmodes Solver

This project is a simple application that allows users to draw shapes on a canvas, and then it calculates and visualizes the eigenmodes of the drawn shape.

## Introduction

The Drawing Eigenmodes Solver is a Python application built using the Tkinter library for the GUI, NumPy for numerical operations, and Matplotlib for visualization. It leverages the concept of eigenmodes to analyze and visualize the behavior of a drawn shape.

## Features

- **Drawing Interface:** Users can draw shapes on a canvas using a mouse.
- **Eigenmode Calculation:** The application calculates and visualizes the eigenmodes of the drawn shape.
- **Sparse Matrix Handling:** Utilizes sparse matrix techniques for efficient computation.
